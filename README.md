# Ramadan 2026 Prayer Times

Mobile-friendly web app to view Ramadan prayer times, Sehri end, and Iftar.

## Highlights
- Today’s Ramadan day shown automatically
- Sehri & Iftar times easy to see
- Full daily prayers (Adhan + Iqamah) + Tarawih
- Jump to any day (1–30)
- Eid al-Fitr prayer times included
